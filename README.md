# QR Code Attendance System

Description
This project is a web-based attendance system that uses QR codes to mark student attendance easily and quickly.

Objective
To automate attendance and reduce manual work using QR technology.

Technologies Used
- Python (Flask)
- SQLite Database
- HTML, CSS
- QR Code Generator

Features
- Generate QR code
- Student attendance marking
- Store attendance in database
- View attendance dashboard

How to Run
1. Install Python
2. Install libraries:
   pip install flask qrcode pillow
3. Run the project:
   python app.py
4. Open browser:
   http://127.0.0.1:5000
